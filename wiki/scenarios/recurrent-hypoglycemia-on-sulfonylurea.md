# Recurrent Hypoglycemia on Sulfonylurea

## Typical Patient

Type 2 diabetes on a sulfonylurea, reporting sweating, tremor, confusion, falls, nighttime symptoms, or glucose readings below target. Risk is higher with CKD, missed meals, older age, alcohol, or insulin co-use.

## First Visit Priorities

| Priority | Action |
|----------|--------|
| Establish severity | Ask about assistance needed, falls, driving events, EMS/ED visits, and nocturnal episodes. |
| Identify pattern | Link lows to meals, fasting, exercise, alcohol, renal decline, or dose timing. |
| De-intensify | Reduce or stop sulfonylurea first when risk is meaningful. |
| Substitute | Use lower-hypoglycemia-risk therapy if A1C still needs treatment. |
| Safety | Teach 15-15 rule, glucagon where appropriate, and when to call. |

## Practical Medication Direction

1. Stop or reduce [[wiki/medications/sulfonylureas|sulfonylurea]] if lows are recurrent or severe.
2. Continue [[wiki/medications/metformin|metformin]] if appropriate.
3. If CKD/HF: use an SGLT2 inhibitor when eligible.
4. If obesity/ASCVD: use GLP-1 RA or GIP/GLP-1 therapy when accessible.
5. If insulin is also used: reduce insulin dose based on pattern and consider CGM.

## Avoid

- Treating hypoglycemia as a patient behavior problem before checking the regimen.
- Keeping sulfonylurea because A1C is "good."
- Adding more drugs without removing the hypoglycemia driver.
