# Insulin Initiation and Intensification

## When To Start Insulin

| Scenario | Practical Move |
|----------|----------------|
| Symptomatic hyperglycemia, catabolism, ketosis concern, or very high glucose | Start insulin urgently and evaluate diabetes type/acute illness. |
| A1C remains high despite appropriate non-insulin therapy | Prefer GLP-1 RA before insulin when clinically appropriate; otherwise start basal insulin. |
| CKD, frailty, or high hypoglycemia risk | Start lower, titrate slower, and simplify. |
| Cost blocks GLP-1/SGLT2 therapy | Use affordable insulin safely with structured monitoring. |

## Basal Insulin Workflow

1. Confirm fasting vs post-prandial pattern from SMBG or CGM.
2. Start basal insulin with a clear titration plan.
3. Continue metformin if appropriate.
4. Reduce or stop sulfonylurea if fasting glucose improves or hypoglycemia occurs.
5. Titrate every 3-7 days using fasting glucose, not A1C alone.
6. If fasting is controlled but A1C remains high, address meals: GLP-1 RA, prandial insulin, diet pattern, or fixed-ratio combination depending on access.

## Intensification Choices

| Problem | Better Next Step |
|---------|------------------|
| High fasting glucose | Titrate basal insulin. |
| Fasting controlled, A1C high | Add GLP-1 RA or prandial/mealtime strategy. |
| Hypoglycemia overnight | Reduce basal dose, check dinner/bedtime pattern, consider CGM. |
| Weight gain on insulin | Add/optimize GLP-1 RA or SGLT2 inhibitor if appropriate. |
| Complex regimen with poor adherence | Simplify before adding more injections. |

## Avoid

- Escalating basal insulin endlessly when fasting glucose is already at goal.
- Keeping sulfonylurea at full dose after insulin or GLP-1 RA improves glucose.
- Starting insulin without teaching hypoglycemia recognition and treatment.
- Using A1C alone in anemia, CKD, pregnancy, recent transfusion, or suspected discordance.
