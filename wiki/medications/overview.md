## How To Use These Pages

1. Start with the patient phenotype: CKD, heart failure, ASCVD, obesity, hypoglycemia risk, pregnancy potential, frailty, cost/access.
2. Pick the page for the drug class you are considering.
3. Check the "do not use / pause" and monitoring sections before dosing.
4. Link back to the relevant algorithm for sequencing.
